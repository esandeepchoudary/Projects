Android Application.
***********************

Step 1. Copy the "app-debug.apk" onto your Android Phone Memory.

Step 2. Locate this file in the phone memory through a file manager.

Step 3. Navigate to Security in Settings and check the option "unknown sources", tap "OK" on prompt.

Step 4. Click on the file to start the installtion.

Step 5. The application is ready to be used after the end of installation.
